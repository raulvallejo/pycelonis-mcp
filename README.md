# PyCelonis MCP Server

**Give your coding agent deep knowledge of the PyCelonis SDK.**

An MCP (Model Context Protocol) server that indexes the full PyCelonis 2.14.2 documentation and exposes it as a tool that coding agents (Claude Code, Cursor, GitHub Copilot) can call directly. Ask any question about PyCelonis and get a grounded answer with source citations — without leaving your IDE.

---

## How It Works

**Step 1 — Ingestion (one time)**
Crawls and indexes all PyCelonis 2.14.2 docs into Pinecone using OpenAI embeddings. Run once before starting the server.

**Step 2 — MCP Server (runs locally)**
Starts a local MCP server that exposes the `ask_pycelonis` tool. Your coding agent calls the tool with a question, retrieves the top matching doc chunks from Pinecone, and gets a grounded answer with source citations generated by Groq.

---

## Supported Agents

- Claude Code
- Cursor
- GitHub Copilot
- Any MCP-compatible coding assistant

---

## Stack

| Layer | Technology |
|---|---|
| Embeddings | OpenAI `text-embedding-3-small` |
| Vector store | Pinecone |
| Generation | Groq `llama-3.3-70b-versatile` |
| RAG chain | LangChain |
| Observability | OPIK by Comet |
| MCP server | MCP Python SDK |
